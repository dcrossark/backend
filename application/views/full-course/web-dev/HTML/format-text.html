Pemformatan Teks HTML

Pemformatan Teks di HTML

Teks dalam dokumen web dapat diformat secara khusus untuk menunjukkan perbedaan dan penekanan terhadap isi dan maksud dari teks tersebut. Perbedaan ditampilkan dalam bentuk huruf tebal, miring, garis bawah, dan lain-lain.

Pemformatan Teks

Format teks didalam HTML

  • Menebalkan huruf (bold)
  • Memiringkan huruf (italic)
  • Garis bawah
  • Teks tercoret
  • Membuat marker
  • Membuat pangkat

Teks Tebal

Teks tebal bisanaya diguanakan untuk memberikan penegasan pada teks tertentu, misalnya seperti judul, subjudul, huruf penting, dll.

Tag yang digunakan untuk membuat teks tebal di HTML adalah tag <b> (bold) . Kamu bebas mau pakai yang mana saja, hasilnya akan sama-sama tebal.
Contoh : 

<!DOCTYPE html>
 <html>
 <head>
    <title> Membuat teks Tebal </title>
 </head>
 <body>

   <p>
      <strong> Teks Formating itu Penting! </strong> Karena
      dapat membuat tulisan terlihat lebih menarik sehingga akan membuat <b> Pengunjung senang </b>
      membacanya.
   </p>
 </body>
</html>

Teks Miring

Teks miring digunakan untuk menegaskan sebuah kata atau istilah baru. Teks miring di HTML dapat kita dibuat dengan <i> (italic) dan juga tag < em> (emphasis).
Contoh : 

<!DOCTYPE html>
<html>
  <head>
       <title> Membuat teks Tebal </title>
  </head>
  <body>
       <p>
       Gunakan <i> Teks Miring </i> untuk memberikan penekanan pada teks
       , sehingga akan <em> menarik perhatian </em> pembaca. Biasanya digunakan
       pada <i> istilah asing </i> atau serapan dari
       <em>bahasa daerah </em>.
  </p>
 </body>
</html>

Garis Bawah

Garis bawah biasanya digunakan untuk menandai teks yang disisipkan atau teks yang memiliki arti penting dibandingkan Teks normal lainnya.
Garis bawah di HTML dapat kita buat dengan tag <u> (underline) dan juga tag < ins> (insert).
Contoh : 

<!DOCTYPE html>
<html>
  <head>
     <title> Garis bawah di HTML </title>
  </head>
  <body>
    <p>
             <u> Text formatting itu penting </u>, karena dapat membuat teks terlihat
            lebih menarik dibandingkan <del> Teks biasa </del>
             <ins> plain text </ins>
     </p>
  </body>
 </html>

Teks Tercoret

Teks tercoret memiliki arti teks yang dihapus. Biasanya untuk memberitahu pembaca bahwa teks tersebut tidak dipakai atau terhapus. Tag untuk membuat teks tercoret di HTML adalah tag <s> (strikethrough) atau bisa juga dengan tag <del> (delete).
Contoh : 

<!DOCTYPE html>
<html>
   <head>
           <title> Garis bawah di HTML </title>
   </head>
   <body>
           <p>
                 Coretlah teks yang tidak <s> dibutuhkan </s>, terpakai, ini bisa memberitahu
                 pembaca tentang perbaikan dari teks tersebut. Kadang juga teks <del> Tercoret </del>
                 <ins> yang dicoret </ins>, diperbaiki dengan menambahkan teks dengan garis bawah.
           </p>
    </body>
</html>

Marker untuk Teks

Marker biasanya digunakan untuk menandai teks yang penting atau kata kunci yang penting. marker di HTML dapat kita buat dengan tag <mark>.

Contoh : 

<!DOCTYPE html>
<html>
   <head>
       <title> Garis bawah di HTML </title>
   </head>
   <body>
       <p>
           Marker biasanya digunakan untuk menandai bagian teks yang penting.
           Kalau di dunia nyata, kita <mark> menggunakan stabilo </mark>
           untuk membuat marker
       </p>
   </body>
</html>

Membuat pangkat

Pangkat digunakan pada rumus. dalam pemangkatan ada dua jenis yang bisa di buat di dalam HTML, yakni angkat yang berada di ata (superscript) dan pangkat di bawan (subscript).
Tag untuk membuat pangkat di HTML adalah tag <sup> dan <sub>.

  • <sup> untuk membuat pangkat di atas
  • <sub> untuk membuat pangkat di bawah

Contoh : 



<!DOCTYPE html>
<html>
   <head>
            <title> Garis bawah di HTML </title>
   </head>
   <body>
            <p>
            Rumus luas persegi adalah S <sup>2</sup>, dimana <i>S</i> adalah
            sisi dari persegi. Lalu O <sub>2</sub> adalah kimia dari oksigen.
           </p>
    </body>
</html>

Teks Preformat

Digunakan untuk menampilkan sesuai dengan dokumen web tersebut dalam editor teks. Jika Anda ingin menambahkan sebuah pantun dalam tampilan web, setiap baris hanyalah memiliki beberapa kata, jika kita menggunakan tag <br> sudah Pasti kita memerlukan beberapa tag tersebut. Nah untuk mempermudah dalam hal itu kita bisa menggunakan tag <pre>.
Contoh : 

<!DOCTYPE html>
<html>
   <head>
        <title> Garis bawah di HTML </title>
   </head>
   <body>
        <pre>
        Kalau suka bermain gitar
        Mainlah yang baik di atas pagar
        Kalau mau menjadi pintar
        Jangan suka malas belajar
        </pre>
   </body>
</html>
