Pengertian Tag HTML dan Fungsinya

Tag dalam HTML

Tag adalah sebuah penanda awalan dan akhiran dari sebuah elemen di HTML. Tag dibuat dengan kurung siku (<...>), lalu di dalamnya berisi nama tag dan kadang juga ditambahkan dengan atribut.
Contoh Beberapa Tag yang ada dalam HTML ialah : <p>, <a>, <body>, <head>, dan sebagainya.

Dalam penulisan Tag selalu berpasangan. Ada tag pembuka dan tag penutup. Tetapi dalam penulisan Tag ada beberapa tag yang tidak memiliki pasangan penutup. Tag penutup ditulis dengan menambahkan garis miring (/) di depan nama tag.

Dalam penulisan HTML ada beberapa Tag-tag yang wajib kita sertakan sebelum memulai pengerjaan, Diantaranya :

  • <!DOCTYPE html> - Untuk mendeklarasikan type dokumen
  • <html> - Tag utama dalam HYML
  • <head> - untuk bagian kepala dari dokumen
  • <title> - Untuk judul web
  • <body> - Untuk bagian body dari dokumen

Tag Head

Tag Head berfungsi untuk menyimpan informasi dari sebuah halaman misalnya tag <title> (Judul dari sebuah Halaman) dan <meta> (pengkodean karakter UTF-8). Penulisan Tag head memiliki tag pembuka <head> dan tag penutup </head> apapun yang di tuliskan di dalam tag head tidak akan ditampilkan di halaman webiste yang kita buat
Contoh penulisan tag head : 

<head>
  isi dari informasi halaman, misalnya isi dari <title> dan <meta>
</head>

Tag Title

Tag title merupakan tag yang digunakan untuk menuliskan judul dokumen HTML. Hasil tag ini akan ditampilkan dalam window caption browser.
Contoh : 

<head>
   <title>Contoh dari Judul Website</title>
</head>

Tag Body

Tag body merupakan bagian terdalam dari sebuah html, dan bagian ini yang akan di tampilkan dalam sebuah website.
Contoh penulisan tag body : 

<body>

    <h1> Contoh dari Judul </h1>

    <p>Contoh dari Paragpaph</p>

</body>

Heading

Heading dimanfaatkan untuk menandai judul topic atau untuk menunjukkan tingkat keberartian dari teks yang akan dituliskan. Ada 6 tingkat heading dalam HTML, dinomori dari 1 sampai dengan 6. Nomor 1 merupakan heading yang terbesar.
Contoh : 

<body>

    <h1> Contoh dari Judul Tag h1 </h1>

    <h2> Contoh dari Judul Tag h2 </h2>

    <h3> Contoh dari Judul Tag h3 </h3>

    <h4> Contoh dari Judul Tag h4 </h4>

    <h5> Contoh dari Judul Tag h5 </h5>

    <h6> Contoh dari Judul Tag h6 </h6>

</body>
