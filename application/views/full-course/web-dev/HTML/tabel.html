Tabel

Membuat Tabel

Tabel digunakan untuk menyajikan data dalam bentuk kolom dan baris. Untuk mengawali pembuatan Tabel di awali dengan tag <table>. Tabel sudah identik dengan baris dan kolom,dalam penulisan baris di dalam tabel bisa gunakan tag <tr>. dan untuk pembuatan kolom Anda bisa menggunakan tag <td>. Td kependekan dari "table data" yang berarti tempat menyimpan data (data cell). Sebuah cell data dapat berupa teks, image, list, paragraf, form, tabel, dll.
Contoh : 

<body>
    <table border="1px solid black">
         <tr>
            <td> Nama </td>
            <td> Jabatan </td>
         </tr>

         <tr>
            <td> Michael </td>
            <td> Founder </td>
         </tr>
    </table>
</body>

Menggabungkan Sel Tabel

Atribut yang digunakan untuk menggabungkan sel tabel adalah rowspan dan colspan :

  • rowspan untuk menggabungkan baris
  • colspan untuk menggabungkan kolom


Contoh : 
<table border="1" Cellpadding="10">

<tr>
          <th rowspan="2" bgcolor="yellow"> Nama </th>
          <th colspan="2" bgcolor="green"> Nilai</th>
</tr>

    <tr>
              <td> UTS </td>
              <td> UAS </td>
    </tr>

    <tr>
              <td> Michael </td>
              <td> 80 </td>
              <td> 75 </td>
    </tr>

    <tr>
              <td> Dimas </td>
              <td> 85 </td>
              <td> 90 </td>
    </tr>

    <tr>
              <td> Rani </td>
              <td> 90 </td>
              <td> 90 </td>
    </tr>

</table>

Cellpadding

Digunakan untuk mengatur padding cell (jarak antara border dengan konten).
Contoh : 

<table border="1" Cellpadding="10">

  <tr>
     <td> Nama </td>
     <td> Jabatan </td>
  </tr>

  <tr>
      <td> Michael </td>
      <td> Founder </td>
  </tr>

</table>

Hasil :

Cellspacing

Digunakan untuk memberi jarak antar cell.
Contoh : 

<table border="1" Cellspacing="10">
    <tr>
           <td > Nama </td>
           <td> Jabatan </td>
    </tr>

    <tr>
           <td > Michael </td>
           <td> Founder </td>
    </tr>

</table>

Hasil :

Background Warna

Untuk memberi background warna pada tabel dengan menggunakan bgcolor.
Contoh : 

<table border="1" Cellpadding="10">
    <tr>
         <td bgcolor="yellow"> Nama </td>
         <td> Jabatan </td>
    </tr>

    <tr bgcolor="#00ff80">
        <td > Michael </td>
        <td> Founder </td>
    </tr>

</table>

Hasil :

Warna Border

Untuk memberi warna pada border dengan menggunakan bordercolor.
Contoh : 

<table border="1" bordercolor="red">
    <tr>
           <td > Nama </td>
           <td> Jabatan </td>
    </tr>

    <tr>
           <td > Michael </td>
           <td> Founder </td>
    </tr>

</table>
