Hyperlink HTML

Hyperlink

Dokumen HTML mempunyai kemampuan untuk memberikan link dari satu teks atau gambar menuju ke dokumen/halaman Lain dalam suatu dokumen. Browser web akan menyorot teks atau gambar yang diidentifikasikan sebagai link dengan warna atau garis bawah untuk menunjukkan bahwa itu adalah hypertext link.
HTML menggunakan tag <a> yang disebut sebagai tag anchor untuk membuat suatu link kepada dokumen lain.
Contoh : 

<body>
  <p>
      <a href="https://www.bercoding.id">Contoh Hyperlink</a>
  </p>
</body>

Hasil :

Membuat link untuk Tab Baru

Untuk membuat link supaya membuka tab baru dapat ditambahkan atribut: target="_blank"
Contoh : 

  <body>
    <p>
          <a href="https://www.bercoding.id" type="_blank">Contoh Hyperlink</a>
    </p>
  </body>

Hasil :

Mailto

Berikut ini adalah contoh link untuk menghubungkan ke sebuah alamat email.
Contoh : 

  <body>
      <p>
            <a  href="mailto:bercoding.id@gmail.com">Contoh Hyperlink mailto </a>
      </p>
  </body>

Image Hyperlink

Hyperlink dengan gambar, dalam menanamkan link di sebuah gambar memang terlihat seperti tidak ada yang berubah tetapi ketika Anda mengklik gambar tersebut halaman secara otomatis akan beralih ke Hyperlink yang sudah anda buat.
Contoh :

<a href="#"> <img src ="https://www.bercoding.id/img/logo.jpg" /> </a>
