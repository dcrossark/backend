List

List Item

List item di gunakan untuk mengelompokkan data baik berurutan (ordered ist) maupun yang tidak berurutan (unordered list).

Ordered List

Untuk membuat list terurut nomor (ordered list). Kita gunakan tag pembuka <ol> dan penutup tag </ol>, sedangkan untuk mendata setiap itemnya menggunakan tag <li> sebagai pembuka dan </li> sebagai penutup.
Contoh : 

<body>
  <ol>
   <li> Contoh 1 </li>
   <li> Contoh 2 </li>
   <li> Contoh 3 </li>
  </ol>
</body>

Hasil :

Ada beberapa tipe yang bisa Digunakan, yaitu :
  • < ol type="a"> = untuk alfabet a, b, c, dan seterusnya
  • < ol type="A"> = untuk alfabet A, B, C, dan seterusnya
  • < ol type="i"> = untuk angka romawi i, ii, iii, dan seterusnya
  • < ol type="I"> = untuk alfabet I, II, III, dan seterusnya

Contoh : 
  <body>

   <h3> List dengan type "a" </h3>

      <ol type="a">
           <li> Contoh 1 </li>
           <li> Contoh 2 </li>
           <li> Contoh 3 </li>
      </ol>

  <h3> List dengan type "A" </h3>
      <ol type="A">
           <li> Contoh 1 </li>
           <li> Contoh 2 </li>
           <li> Contoh 3 </li>
      </ol>

<h3> List dengan type "i" </h3>
      <ol type="i">
           <li> Contoh 1 </li>
           <li> Contoh 2 </li>
           <li> Contoh 3 </li>
      </ol>

<h3> List dengan type "I" </h3>
      <ol type="I">
           <li> Contoh 1 </li>
           <li> Contoh 2 </li>
           <li> Contoh 3 </li>
      </ol>

  </body>

Hasil :

Unordered List

Untuk membuat list tidak terurut nomor (unordered list), kita gunakan tag pembuka<ul> dan penutup tag </ul>, sedangkan untuk mendata setiap itemnya menggunakan tag <li> sebagai pembuka dan </li> sebagai penutup.
Contoh : 

<body>
    <ul>
         <li> Contoh 1 </li>
         <li> Contoh 2 </li>
         <li> Contoh 3 </li>
    </ul>
</body>

Hasil :

Ada 3 tipe yang bisa kita gunakan yaitu square, disc, circle.
Contoh : 
<body>

<h3> Unordered dengan type "cirle" </h3>

  <ul type="cirle">
       <li> Contoh 1 </li>
       <li> Contoh 2 </li>
       <li> Contoh 3 </li>
  </ul>

<h3> Unordered dengan type "disc" </h3>
  <ul type="disc">
       <li> Contoh 1 </li>
       <li> Contoh 2 </li>
       <li> Contoh 3 </li>
  </ul>

<h3> Unordered dengan type "square" </h3>
  <ul type="square">
       <li> Contoh 1 </li>
       <li> Contoh 2 </li>
       <li> Contoh 3 </li>
  </ul>
</body>
